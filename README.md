# AuraMusic Messages

This repository contains messages, announcements, and updates for the AuraMusic app.

## Structure

```
├── messages.json          # Message index
├── announcements/         # General announcements
├── updates/              # App updates and releases
├── maintenance/          # Maintenance notices
└── alerts/              # Important alerts
```

## Adding New Messages

1. Create a new `.md` file in the appropriate folder
2. Add entry to `messages.json` with the GitHub raw URL
3. Commit and push changes

## Message Types

- `announcement` - General news and features
- `update` - App updates and changelogs
- `maintenance` - Server maintenance notices
- `alert` - Important alerts and notices
